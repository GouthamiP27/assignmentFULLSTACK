# Student Report Card System

This is a simmple CLI project made using JavaScript and Node.js. The program takes a student's name and marks as input from comman d line, calculates the average marks and gives a letter grade.

## Files
- reportcard.js - Main file of the project
- student.js - Contains Student class and related methods

## Features
- Takes student name and marks as input
- Calculates average score
- Assigns grade based on average
- Displays a formated report card

## How to Run

```bash
node reportcard.js Rahul 85 90 78
